I wanted to learn to program well in C++ -- interesting things hapen in C++, and I've also wanted to build up a thorough understanding of the fundamentals of machine learning; so of course I thought I'd kill two birds with one stone and learn C++ while writing a machine learning library from scrach.

I'm not coming in completely cold: I used C++ for a simulation project on my engineering degree -- but it was the work of a self-taught engineering student, and it shows. I've also studied the fundamentals of machine learning when I studied computer science, but need to blow off the cobwebs.

I've started off making some utilities, point classes (I'm working with numeric coordinates for the time being). I've move on to implementing simple things - first port of call was the perceptron of course. Next step was to make a simple multiclass version (really a tagger).

I'm currently working on implementing a basic SVM, and on addding some evaluation tools.
