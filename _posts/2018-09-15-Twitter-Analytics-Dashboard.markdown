I wrote this for my MSc dissertation. It is an electron app that provides some generic tools for gathering and analysing a large set of tweets. It contains the normal NLP text cleaning functions, a function to gather tweets through Twitter's API, plotting, and some ML tools.

Update: the Twitter API used here was depricated.
